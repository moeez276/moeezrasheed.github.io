## ğŸ¨ Customization

### Colors
Modify CSS custom properties in `enhanced-3d.css`:
```css
:root {
    --primary-bg: #0a0a0a;
    --neon-cyan: #00f5ff;
    --neon-blue: #0066ff;
    /* Add your custom colors */
}
```

### Animations
Adjust GSAP timelines in `portfolio-3d.js`:
```javascript
gsap.timeline()
    .from('.hero-avatar', { 
        scale: 0, 
        rotation: 360, 
        duration: 1.5 
    });
```

### Content
Update content directly in `index_3d.html` while preserving the structure.
